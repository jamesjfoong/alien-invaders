# Alien Invaders

A simple HTML5 game where you defend Earth from alien invaders.

## How to Play

1. Open \`index.html\` in your web browser.
2. Use left and right arrow keys to move.
3. Press spacebar to shoot.
4. Destroy all aliens to win!
